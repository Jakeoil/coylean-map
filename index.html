Coylean Map Program

A Coylean Map is a unique pattern produced by subdividing a square with vertical and horizontal lines in a specified fashion. The lines are drawn so as to stop and continue where they meet a perpendicular line. The finished pattern contains similar copies of previous iterations making this pattern a fractal.

This pattern was discovered by Jeff Coyle. During the early 70s many examples were drawn by hand. The first computer implementation was done on a Commodore PET using their character set followed by a more graphical version on the Apple II. This implementation is a web port of an Android app ported from a C# version done in .NET.

Context: A Coylean Map is a fractal grid pattern generated by binary bisection of a square using horizontal and vertical lines with seniority-based interruption rules. A 4x4 section of a Coylean Map is fully described by two 3-bit binary sequences — one encoding which of the 3 interior vertical lines are present (right-priority), one encoding which of the 3 interior horizontal lines are present (down-priority). This gives 8×8 = 64 combinations.
Step 1: Render all 64 combinations as a labeled 8×8 grid — rows indexed by down-priority (0–7), columns by right-priority (0–7). Each cell shows a small canvas drawing of that 4×4 Coylean section with correct seniority interruption: a vertical line of higher priority passes uninterrupted through a horizontal of lower priority and vice versa. The priority of each interior line is its position in the ruler/2-adic valuation sequence.
Step 1a: Note that there is another set that follows hozontal seniority (later step)
Step 2: Classify all 64 combinations under the action of the dihedral group D4 (4 rotations × 2 reflections = 8 symmetries). Group them into equivalence classes. For each class identify the symmetry subgroup of the representative (which D4 symmetries map it to itself). Display the unique canonical representatives in a second labeled grid, annotating each with its D4 symmetry group name (e.g. trivial, Z2-rotation, Z2-mirror-H, Z2-mirror-V, Z2-mirror-D1, Z2-mirror-D2, Z4, D2, D4) and the count of elements in its equivalence class.

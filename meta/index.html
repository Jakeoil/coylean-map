Meta

Sketches and side experiments — not maps themselves, but tools and themes built around the project.

How to Draw a Coylean Map

A short walkthrough of the algorithm — priority, reaction, propagation — plus how to translate the boolean output into Unicode box-drawing characters. Comes with a Node renderer (draw-coylean.mjs), a Line Style Variants page, and a Proper Coylean Square (16×16, seeded from NW context) in the same folder.

drawPipeJunction Tester

Live tester for the orthographic red/blue pipe-junction drawing function. Adjust x, y, size, blueD, and redD with sliders and watch the result update in real time. Two backends with identical geometry:

canvas Canvas tester svg SVG tester

Arrow Thickness Tester

Interactive tester for calibrating per-priority arrow presets — adjusts shaft width, head length, head width, nose offset, and cap margin. Used to tune the arrow presets in coylean-explorer/src/display/arrows.js.

OKLCH Color Theme

A palette sketch in the OKLCH color space exploring outline, shadow, base, highlight, and glow tones across red, amber, green, blue, and violet hues for use in the UI.

Ruler Grid Sphere

A spherical projection of the ruler grid using Mercator latitude spacing, with zoom controls — the globe analogue of the planar Coylean ruler subdivisions.

Ruler Grid Horn Torus

The same dyadic ruler grid on a horn torus, where the tube circumference 2πr equals the major circumference 2πR so the inner hole degenerates to a single point.

Sliding Ruler Test Page

Interactive harness for the canvas-based SlidingRuler control from lg-remote/volume-ruler-control — live controls for min, max, value, height, visibleRange, all seven color slots, and a JSON labels override. Light-mode preset by default; also ships dark, paper, high-contrast, and labeled presets for comparison.

3D Lattice

A Three.js workspace for experimenting with 3D renderings of Coylean maps. Currently a hello-world scene; lattice viewers will live here.

4D

Four-dimensional sketches — the tesseract viewer and the hyperdiagonal lattice viewer.

Big Map

Lazy tile reconstruction over very large Coylean universes. Sweep the propagation in K×K blocks, keep only the seams, rebuild any tile on demand. Includes a pan/zoom explorer whose universe grows southeast as you pan past the edge, with dyadic gridlines thickening by priority.

Sliding Ruler

Test harness for the canvas-based SlidingRuler control from lg-remote/volume-ruler-control. The library source lives in ./volume-ruler-control/ alongside its example.html and README.md.

Interactive Test Page

Live controls for every constructor option — min, max, value, height, visibleRange, all seven color slots, and a JSON labels override. Starts in the light-mode preset; switch presets (default dark, light, paper, high contrast, labeled) to compare. Imports the library straight from jakeoil.github.io/lg-remote.

Library’s Own Examples

The upstream example.html shipped with the library — direct class usage plus two web-component examples (temperature, labeled). Useful as a baseline while tweaking the test page above.

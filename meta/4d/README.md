# 4D

Four-dimensional sketches under `meta/`.

- `index.html` — landing page linking to the two viewers below.
- `tesseract.html` — three.js tesseract viewer (rotations in xw/yw/zw,
  2×2×2×2 grid, axis overlays, shared 2-faces, slice walls, cell markers).
- `hyperdiagonal-lattice.html` — 2D projection of the 4D integer lattice
  with hyperdiagonal edges to all 16 (±1,±1,±1,±1) neighbors.
- `tesseract-slice-test.js` — scratch script for tesseract slice geometry.
- `cube-cuts.html` — animated 3D simulation of the recursive
  corner-bisection scheme (3 cuts → 4 cells with sizes ½, ¼, ⅛, ⅛).
- `cube-cuts-plan.md` — design notes for that page and its 4D successor.

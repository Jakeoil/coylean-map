Coylean Glyphs — Stained Glass

Every Coylean map is a substitution tiling: each glyph expands into a 2×2 of glyphs, forever. Here each glyph orbit gets an OKLCH hue, and the map is colored by compositing those hues all the way down the substitution — alpha falls off with depth, so the coarse glyph sets the base color and finer scales add texture. On the anchor family the result is exactly self-similar. Built on the renderer-agnostic orbit-color.js engine.
